Unidades de Medida en CSS

Explora las diferentes formas de medir tamaños y espacios en tus páginas web usando CSS.

¿Qué son las unidades de medida?

Las unidades de medida en CSS se utilizan para definir el tamaño de los elementos, márgenes, paddings, fuentes y otros aspectos visuales de una página. Estas unidades pueden ser absolutas o relativas dependiendo del contexto en el que se usen.

Unidades Absolutas

Las unidades absolutas tienen un valor fijo que no cambia según el entorno o el dispositivo. Son ideales para medios impresos o cuando se necesita una medida precisa.

  • pt - Puntos
  • in - Pulgadas
  • cm - Centímetros
  • mm - Milímetros
  • px - Píxeles

Unidades Relativas

Las unidades relativas dependen del tamaño de otros elementos, del viewport o del contenedor. Son más flexibles y útiles para diseños adaptativos.

  • em - Relativo al tamaño de fuente del elemento padre
  • rem - Relativo al tamaño de fuente del elemento raíz
  • % - Porcentaje del contenedor
  • vw - Porcentaje del ancho del viewport
  • vh - Porcentaje del alto del viewport
  • vmin - Mínimo entre ancho y alto del viewport
  • vmax - Máximo entre ancho y alto del viewport

Conclusión

El uso correcto de unidades de medida permite crear sitios web responsivos, accesibles y visualmente coherentes. Las unidades absolutas son útiles para medidas exactas, mientras que las relativas ofrecen adaptabilidad según el dispositivo o entorno.
